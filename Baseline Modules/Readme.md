## Baseline Modules

Installed after the System sets and provide the lightest variation to the baseline vanilla experience while still delivering many improvmements.

### Baseline Server

**Mod Notes**
-  The Cloth Config API, which is installed Server-side in this module to support SpaghettiTrees, was previously installed Client-side in the System Client module.  While it is possible to install it as part of the System Server module, there are no mods in that module which require it as a dependency, hence it doesn't appear server-side until now.  Fabric API and MidnightLib have much the same issue.
- If manually downloading mods, care needs to be taken with Villager Names.  If Minecraft Comes Alive is also being installed (General SMP set), the Villager Names mod needs to be [Villager Names (Fabric Version)](https://www.curseforge.com/minecraft/mc-mods/villager-names-fabric-version), which is the mod listed below.  The alternate naming mod is [Villager Names (Fabric)](https://www.curseforge.com/minecraft/mc-mods/villager-names-fabric), which causes client crashes whenever a player enters a village if Minecraft Comes Alive is installed.  The other mod is mentioned because if MCA is uninstalled, the alternate villager name mod provides a broader selection of names to a broader selection of villager and near-villager characters and mobs (e.g. golems).

### Baseline Client

**Mod Notes**
- Simple Voice Chat breaks on earlier versions of OS X due to older supported libraries.  Explain and provide.  OS X systems will also have difficulty activating microphone support through the vanilla Minecraft client (due to the system level SIP).  There are described workarounds available, but the simplest is to switch launchers to PolyMC, which will request the appropriate permissions during game launch.
- If manually downloading mods, care needs to be taken with Villager Names.  If Minecraft Comes Alive is also being installed (General SMP set), the Villager Names mod needs to be [Villager Names (Fabric Version)](https://www.curseforge.com/minecraft/mc-mods/villager-names-fabric-version), which is the mod listed below.  The alternate naming mod is [Villager Names (Fabric)](https://www.curseforge.com/minecraft/mc-mods/villager-names-fabric), which causes client crashes whenever a player enters a village if Minecraft Comes Alive is installed.  The other mod is mentioned because if MCA is uninstalled, the alternate villager name mod provides a broader selection of names to a broader selection of villager and near-villager characters and mobs (e.g. golems).
- AmbientSounds and CreativeCore have a parity split between the Curseforge and Modrinth versions, with Curseforge having the latest releases.
